# Asteroids

## Installing and running local server

1. `npm i` <br>
2. `npm run start` <br>
3. Open `http://127.0.0.1:8080/` in browser<br>
4. Press `Enter` to begin/pause
5. Press `R` to restart

## Handling

- `w` or `arrow up` - accelerate
- `a` or `arrow left`, `d` or `arrow right` - turn ship
- `space` - shoot
